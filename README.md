# Artificial Intelligence in Business Process Management

Minsky brings system dynamics and monetary modelling to economics. PySD is a simple library for running System Dynamics models in Python, with the purpose of improving integration of Big Data and Machine Learning into the SD workflow. Stock & Flow is a visual systems dynamics simulator with AI-powered model generation and an MCP server for AI agent integration. SAP Cloud SDK for AI is the official Software Development Kit (SDK) for SAP AI Core, SAP Generative AI Hub, and Orchestration Service. Archi is a free, open source, cross-platform tool and editor to create ArchiMate models. Microsoft Dynamics 365 Business Central. Activiti is a light-weight workflow and Business Process Management (BPM) Platform. Orienteer is Business Application Platform. Bonita empowers organizations with cutting-edge BPA. Open Source Robotic Process Automation Software. Apache OFBiz is an open-source product for business process automation. At the core of the Flowable products are fast, modern process and case management and rules engines that are open source and fully support the BPMN, CMMN, and DMN open standards. Airbyte is a data integration platform. Zato is a platform that lets you automate, integrate and orchestrate business systems, APIs, workflows as well as hardware assets in industries such as airports, defense, health care, telecommunications, financial services and more. Apache NiFi is a visual data flow based system. n8n is a workflow automation platform. Oryx 2 lambda architecture on Apache Spark, Kafka for real-time large scale machine learning. Business Simulation Library (BSL) is a Modelica library for modeling and simulation of dynamical systems. LunaSim is a Lightweight, Web-Based, System Dynamics Modelling Software. Simlin is an open source tool for creating and editing System Dynamics models

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
